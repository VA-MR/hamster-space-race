# Audio Files

Place the following audio files in this directory:

1. **cosmic-gameplay.mp3** - Looping ambient space music for gameplay
2. **mission-complete.mp3** - Short triumphant sound when reaching Earth (2-5 seconds)
3. **celebration.mp3** - Upbeat celebratory music for the results page
